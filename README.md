# 華容放曹 — 合併高效版（BFS + A*） v1.0.7

- **時間區域**：頁面「最後更新時間」與程式常數均以 **台灣時間 (GMT+8)** 固定呈現（打包時寫入）。
- **功能**：
  - **BFS**（關羽吃寶箱，保證最短）
  - **A***（曹操脫逃，高效啟發式）
- **效能顯示**：**秒數**（到小數點下兩位）：讀檔 / 求解 / 總計；節點統計（展開 / 訪問 / 峰值）。
- **合法性**：目標區塊必須是空格，禁止重疊（同字母不同棋子不可穿越）。
- **寶箱遮蔽規則（頁面副標顯示）**：小寫表示棋子覆蓋寶箱：`c(C)` / `g(G)` / `h(H)` / `v(V)` / `s(S)`。

## 使用方式
1. 以瀏覽器開啟 `index.html`
2. 載入範例或貼入盤面，點選：
   - **開始求解（BFS：關羽吃寶箱）**
   - **開始求解（A*：曹操脫逃）**
3. 檢視步驟清單、逐步回放；可按 **下載解答（TXT）** 產生 `solution_moves.txt` / `solution_path.txt`  
   （檔尾附：作者 **JoshuaYen**、版本（常數：v1.0.7）、更新時間（使用者本機））
