
# 華容放曹 — 合併高效版（BFS + A*） v1.0.4

- **功能**：BFS（關羽吃寶箱，保證最短）／A*（曹操脫逃，高效）。
- **效能顯示**：**秒數**（到小數點下兩位）：讀檔 / 求解 / 總計；節點統計（展開 / 訪問 / 峰值）。
- **合法性**：目標區塊必須是空格，禁止重疊（同字母不同棋子不可穿越）。
- **寶箱遮蔽規則**：寶箱以 `X`（或 `x`）標記；若被棋子遮蓋，該格以小寫表示：`c`(C)、`g/h`(G/H)、`v`(V)、`s`(S)。解析器會將小寫覆蓋轉回對應的大寫並保留座標。

## 使用
1. 開啟 `index.html`
2. 按 **開始求解（BFS：關羽吃寶箱）** 或 **開始求解（A*：曹操脫逃）**
3. 檢視步驟、逐步回放；可下載 `solution_moves.txt` 與 `solution_path.txt`（檔尾附：作者 JoshuaYen／版本／更新時間）
