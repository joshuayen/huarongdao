
# 華容放曹 — 合併高效版（BFS + A*） v1.0.3

- **功能**：
  - **BFS**（關羽吃寶箱，保證最短）
  - **A\***（曹操脫逃，高效啟發式）
- **效能顯示**：**秒數**（到小數點下兩位）：讀檔 / 求解 / 總計；並含節點統計（展開 / 訪問 / 峰值）。
- **合法性**：移動目標區塊必須是空格，禁止重疊（同字母不同棋子不可穿越）。
- **寶箱覆蓋規則**：
  - 寶箱以 `X`（或 `x`）標記目標位置。
  - **若被其他棋子遮蓋，該格以小寫表示**：`c`(C) / `g`(G) / `h`(H) / `v`(V) / `s`(S)。
  - 解析器會把小寫覆蓋轉回對應的大寫棋子，並保留寶箱座標用於判斷。

## 使用
1. 以瀏覽器開啟 `index.html`
2. 載入範例或貼入盤面，點選：
   - **開始求解（BFS：關羽吃寶箱）**
   - **開始求解（A*：曹操脫逃）**
3. 檢視步驟清單、逐步回放；可按 **下載解答（TXT）** 產生 `solution_moves.txt` / `solution_path.txt`
