
# 華容放曹 — 合併高效版（BFS + A*）

- **功能**：BFS（關羽吃寶箱，最短）／A*（曹操脫逃，高效）
- **效能顯示**：**秒數**（到小數點下兩位）：讀檔 / 求解 / 總計；節點統計（展開/訪問/峰值）
- **合法性**：移動目標區塊必須是空格，禁止重疊（同字母不同棋子也不可穿越）

## 使用
1. 以瀏覽器開啟 `index.html`
2. 載入範例或貼入盤面，點選：
   - **開始求解（BFS：關羽吃寶箱）**
   - **開始求解（A*：曹操脫逃）**
3. 檢視步驟清單、逐步回放；可按 **下載解答（TXT）** 輸出 moves/path 檔（檔尾附作者/版本/更新時間）
